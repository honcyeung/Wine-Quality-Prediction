# Wine-Quality-Prediction
A python machine learning project to predict wine quality with chemical components.

Features:
- fixed acidity
- volatile acidity
- citric acid
- residual sugar
- chlorides
- free sulfur dioxide
- total sulfur dioxide
- density
- pH
- sulphates
- alcohol

Output variable (based on sensory data): 
- quality (score between 0 and 10)
